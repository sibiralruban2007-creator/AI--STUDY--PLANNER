# AI-Powered Study Planner

## Project Description
AI-Powered Study Planner is a web-based application that helps students create personalized study schedules based on subjects and available study hours.

## Technologies Used
- HTML
- CSS
- JavaScript
- Prompt Engineering

## SDG
SDG 4 – Quality Education

## Output
Generates a personalized study plan based on user input.

## Conclusion
This project helps students manage their study time effectively and improve learning productivity.
